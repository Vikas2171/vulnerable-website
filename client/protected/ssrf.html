Server-Side Request Forgery (SSRF)

What is SSRF?

SSRF allows an attacker to make the server send requests to internal or external systems, bypassing access controls like firewalls.

How to exploit:

  • Try an external service: http://httpbin.org/ip
  • Try to access the application server itself: http://localhost:3000
  • Try to access a local file: file:///etc/passwd (may not work on all systems)
