What is SQL Injection?

SQL Injection occurs when user input is directly concatenated into SQL queries, allowing an attacker to manipulate the database.

How to exploit:

  • Try a normal login: username=admin, password=admin123
  • Try a classic bypass: username=' OR '1'='1' -- , password=anything
  • Try to log in as a specific user: username=admin' -- , password=anything
