Reflected Cross-Site Scripting (XSS) via GET

What is Reflected XSS?

The server reads data from the URL and includes it in the response page without sanitization. The payload is visible in the address bar.

How to exploit:

  • Craft a URL with a script payload.
  • Example: /xss-get-vulnerable?search=<script>alert('XSS')</script>
