Reflected Cross-Site Scripting (XSS) via POST

What is Reflected XSS via POST?

This vulnerability occurs when the server takes user input from a form submission and includes it in the response page without proper sanitization.

How to exploit:

  • Try a normal comment: This is great!
  • Try a script injection: <script>alert('XSS via POST')</script>
  • Try an image-based injection: <img src="x" onerror="alert('Hacked!')">
